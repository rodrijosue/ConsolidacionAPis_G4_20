Funciones de los Formularios

Socios:

Articulos:

El formulario de Articulos tiene la funcionalidad de poder agregar los articulos detallando su descripcion, unidad, costo, precio, si aplica al ISV, el porcentaje de este y el Id del socio al que esta ligado este producto. Podemos actualizar los productos previamente ingresados y tambien eliminarlos en caso de que se retire este producto de la tienda o punto de venta, estas dos ultimas funciones pasan por la API en la que se obtiene un articulo especifico ya sea para eliminar o actualizar el articulo que se selecciona.

Pedidos:

Este formulario sirve para mostrar, ingresar, actualizar, eliminar los registros de los pedidos que previamente son ingresados en la base de datos, haciendo uso de nuestras APis REST que desarrollamos anteriormente en PHP.
